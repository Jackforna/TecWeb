The SMM project was developed as a project for the Web Technologies course, taught by Prof. Vitali Fabio at the University of Bologna. The purpose is to create an application for managing a Squealer account, a social messaging network, which seeks to enter the market as an alternative to Twitter. The application was developed using the Angular framework, and interfaces with a server developed in Node.js. The development tools used are:
  • Visual Studio Code
  • GitHub
  • Node.js
  • Angular
  • Express
  • MongoDB
  • Docker
Angular Material components were used for graphics, and Chart.js was used for graph management, and Google Fonts services were used for fonts and logos.
